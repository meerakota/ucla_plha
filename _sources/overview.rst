
Overview
=======================

The **ucla_plha** Python package performs probabilistic liquefaction hazard analysis (PLHA) to quantify the 
annual down-crossing rate (or alternatively, the annual rate of non-exceedance) of factor of safety of liquefaction. It was created
to assess the influence of sea level rise on liquefaction in coastal areas of California. We are sharing the code because it may 
be useful for other researchers studying liquefaction, or for any earthquake problem for which a fragility function is represented by
a lognormal distribution function. Some theory is provided here to explain how the **ucla_plha** package performs its calculations, 
and how it differs from other PLHA codes.
